# Arduino Remote Lab Project
This project is about an Arduino Remote Lab.
It is initiated by A.Benhamouda and L.Mezai.
It has two main parts: Hardware and Software.
Here, we just present the software part.
It includes three servers: Webserver that offers GUI, the WebCam Server that return permanently a list of connected WebCam and the Arduibo Boards Server that retrun permanently the list of the Arduino connected boards.
